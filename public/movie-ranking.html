The Best Movies According to Billy

My top 3 movies of all time.

One Flew Over The Cuckoo's Nest

This is a great movie from 1975, starring Jack Nicholson. It follows Randall MacMurphy's life as a criminal in a mental hospital

Napoleon Dynamite

This is a cult classic, set in a distopian reality about the struggles of teenage life.

The Green Mile

This is a Tom Hanks masterpiece set in a 1930s, death row prison.
